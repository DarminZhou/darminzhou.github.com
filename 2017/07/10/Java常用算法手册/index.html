背包问题–动态规划

背包问题具体例子：假设现有容量10kg的背包，另外有3个物品，分别为a1，a2，a3。
物品a1重量为3kg，价值为4；
物品a2重量为4kg，价值为5；
物品a3重量为5kg，价值为6。
